# Object-Orientated-Programming

# Java Projects Repository

📌 Overview
This repository serves as a collection of Java projects developed for coursework, practice, and portfolio building.  
Each project demonstrates different  object-oriented programming concepts , algorithms, and problem-solving techniques.

 

📂 Repository Structure
Projects are organized into separate folders, each containing its own `.java` files and supporting resources such as the corresponding pdf files

- `Project1_PropertyManagement/` (estate agency system using OOP principles).
- `Project2_Point_of_sale_GUI/` (Java Swing Point of Sale system for managing stock, purchases, and sales reporting).

🛠️ Technologies
-  Language:  Java  
-  IDE:  BlueJ (but compatible with any Java IDE such as IntelliJ IDEA, Eclipse, or NetBeans)  
-  Version Control:  Git & GitHub  

 

▶️ How to Run
1. Clone or download this repository.
2. Navigate to the project folder you want to run.
3. Open the `.java` files in your IDE (BlueJ recommended for coursework).
4. Separate and compile all classes.
5. Run the `main` class (usually named `Test...` or `Main`).

 

📖 Learning Outcomes
- Apply  object-oriented programming principles  (encapsulation, inheritance, polymorphism).
- Work with  file input/output  in Java.
- Build structured, reusable, and maintainable code.
- Practice  problem-solving  and algorithm design.
